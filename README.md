Минималистичное веб‑приложение "текстовый редактор", позволяющий сохранять текст в форматах .txt, .docx и .doc.\
Приложение написано на Python 3.12 + Flask + gunicorn и развёртывается в Docker‑контейнере.\
Главные возможности:
* Веб‑интерфейс для редактирования текста
* Сохранение текста в 3 формата:
   .txt – обычный текст
   .docx – Microsoft Word 2007+
   .doc – старый формат через LibreOffice
* Тесты (Pytest) покрывают основные сценарии
* CI/CD в GitLab: сборка, тесты, деплой на удалённый Docker‑хост
Тесты написаны на pytest. Они проверяют:
* Корректный рендер главной страницы
* Возможность сохранения в каждом формате
* Неправильные входные данные (пустой текст, неизвестный формат)
В CI пайплайне тесты запускаются в Docker‑контейнере (docker:27) внутри job‑а test.
Как работает данный pipeline:
* Build – собирается Docker‑образ, сохраняется в tar‑архив и отдаётся в artifacts.
* Test – берётся артефакт, загружается в Docker‑конвейер, запускается контейнер и проверяются тесты.
* Deploy – только в ветке main. Артефакт копируется на удалённый сервер по SSH, контейнер перезапускается.
Переменные окружения (PROD_HOST, PROD_USER, SSH_PRIVATE_KEY) задаются в настройках проекта GitLab → CI /CD → Variables.
index.html использует простую форму. В качестве стилей лежит один файл style.css
Вы можете легко изменить внешний вид – достаточно поменять style.css и пересобрать Docker‑образ.
