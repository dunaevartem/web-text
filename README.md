Данное веб-приложение написано на языке Python и с помощью написанного пайплайна gitlab-ci.yml билдится, тестируется (test_app.py если добавить в папку test другие тесты то они поочередно будут выполняться) и деплоится на физ./вирт. сервер в виде контейнера докера.\
\
Для корректной отработки пайплайна необходимо осуществить ssh-соединение между раннером и сервером для деплоя, этот ssh-ключ будет скопирован и отправлен с раннера в созданный им контейнер докера. В гитлабе необходимо задать переменную SSH_PRIVATE_KEY\
